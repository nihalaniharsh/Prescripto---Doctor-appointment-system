## 🏥 Full Stack Doctor Appointment Booking System (MERN)

A full-stack web application built using the MERN stack (**MongoDB, Express.js, React.js, Node.js**) that enables seamless appointment booking between patients and doctors.

### 🔑 Key Features

* **Multi-Level Authentication**

  * 👤 Patient Login – Book and manage appointments
  * 🩺 Doctor Login – View appointments, track earnings, update profile
  * 🛠️ Admin Panel – Manage doctors and appointments

* **Appointment Management**

  * Patients can browse doctors and book appointments
  * Doctors can accept, complete, or cancel appointments
  * Admin can monitor all bookings

* **Doctor Dashboard**

  * View total earnings
  * Track appointments
  * Update profile details and availability

* **Admin Dashboard**

  * Add/manage doctors
  * View all appointments
  * Control system operations

* **Online Payment Integration**

  * Secure payment using Razorpay
  * Users can pay appointment fees online

### ⚙️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT
* **Payments:** Razorpay
* **Deployment:** Render

### 🚀 Use Cases

* Can be used by hospitals or individual doctors
* Suitable for real-world applications
